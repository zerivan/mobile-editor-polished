# Editor Mobile (Expo)

Editor de imagens simples e leve feito com React Native + Expo.

## Instalar e rodar
```bash
npm install -g expo-cli
npm install
expo start
```

## Build nativo
```bash
npm install -g eas-cli
eas build --platform android
eas build --platform ios
```
